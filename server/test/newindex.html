Available Routes

This page displays a hardcoded list of API endpoints.

  • /api (API status and location)

  • /api/v1/users (User-related routes)

    • Open

      • /signup
      • /login
      • /logout
      • /isloggedin

    • Protected

      • /resendotp
      • /verify
      • /updateMyPassword
      • /me
      • /updateMe
      • /deleteMe

    • Admin

      • /users (GET, POST)
      • /users/:id (GET, PATCH, DELETE)

  • /api/v1/account (Account-related routes)

    • Protected

      • /(POST): Add a new trading account
      • /myaccounts (GET): Get user's trading accounts
      • /balance/:id (PATCH) (Commented Out): Update account balance
      • /(DELETE) (Commented Out): Delete a trading account

  • /api/v1/trades (Trade-related routes)

    • Protected

      • /place (POST): Place a new trade
      • /close (POST): Close an existing trade
      • /me (GET): Get user's own trade information

  • /api/v1/forex (Forex-related routes)

  • /api/v1/feedback (Feedback-related routes)

    • Open

      • / (GET): Get all feedback (potentially public)
      • / (POST): Create new feedback (potentially requires authentication)

  • /api/v1/gene (Genealogy-related routes)

    • Protected

      • /tree (GET): Get tree below the user
